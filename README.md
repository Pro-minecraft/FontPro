# FontPro
Прежде чем установить приложение Вы должны убедится что:
1. Ваша ОС - это windows 10/11;
2. У вас на устройстве установлен Python 3.6 и выше;
3. Вы должны установить модуль firebase_admin для корректной работы приложения через командную строку(cmd), делается это через команду pip install firebase_admin.

Запуск приложения происходит через запуск файла main.py
